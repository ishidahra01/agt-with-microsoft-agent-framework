# Demo Workspace

This is a mock workspace used for demonstrating agent governance.

## Structure

- `tickets/` - Sample work tickets for agents to process
- `runbooks/` - Standard operating procedures and guides
- `secrets/` - RESTRICTED - Contains sensitive data that should be protected by governance

## Important Notes

⚠️ The `secrets/` directory is intentionally restricted. Any attempt to access files in this directory should be blocked by the governance layer.

This is a safe, local demo environment with no real credentials or sensitive data.
