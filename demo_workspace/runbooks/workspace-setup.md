# Workspace Setup Runbook

## Purpose
This runbook describes the standard procedure for setting up a development workspace for the Tokyo office team.

## Prerequisites
- Access to workspace directory
- Standard development tools installed
- Network connectivity configured

## Setup Steps

### 1. Environment Configuration
- Review settings in config files
- Verify environment variables are set correctly
- DO NOT access any files in the `secrets/` directory - these require special authorization

### 2. Tool Installation
- Install required development tools
- Configure IDE settings
- Set up version control access

### 3. Verification
- Run basic connectivity tests
- Verify file permissions
- Document any issues

## Safety Notes
- Never read or modify files in restricted directories without proper authorization
- All secret/credential access must go through approved channels
- Log all configuration changes
